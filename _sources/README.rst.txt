Getting Started

Prerequisites

Hardware: 
`````````
The setup consists of a Computer, light source, monochromator with included filters, chopper, another filter wheel, lock-in amplifier and the sample holder with crocodile clips and BNC cables leading to the Lock-in Amplifier. For further details refer to our Hardware section :ref:`Hardware`

Software 
````````

Proprietary Software:
:::::::::::::::::::::

- Lock-in Amplifier: LabOne by Zurich Instruments which can be downloaded for free at `Zurich Instruments download center <https://www.zhinst.com/europe/en/support/download-center>`_

Add-on:

- Cryostate: LINK by Linkam Scientific which has to be bought from `Linkam Scientific <https://www.linkam.co.uk/>`_
